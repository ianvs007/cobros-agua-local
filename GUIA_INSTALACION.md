# 💧 Guía de Instalación - Sistema de Cobros de Agua

Sigue estos pasos para instalar y ejecutar el sistema en una computadora nueva.

## 📋 Requisitos Previos

Antes de empezar, asegúrate de tener instalado:
1. **Node.js**: Descárgalo e instálalo desde [nodejs.org](https://nodejs.org/). 
   - Se recomienda la versión **LTS** (la más estable).
   - Durante la instalación, presiona "Next" en todo sin cambiar nada.

## 🚀 Pasos para la Instalación

1. **Copiar el Proyecto**: 
   - Copia la carpeta completa `cobros_agua` a la nueva computadora (puedes usar un USB o enviarla comprimida).
   - Pon la carpeta en un lugar fácil de encontrar, como el Escritorio o la carpeta de Documentos.

2. **Ejecutar el Sistema**:
   - Entra a la carpeta `cobros_agua`.
   - Busca el archivo llamado **`iniciar_sistema.bat`**.
   - Haz doble clic sobre él.

## 🛠️ ¿Qué pasará al ejecutar el archivo?

1. **Configuración Automática**: La primera vez que lo abras, verás una pantalla negra (terminal) que dirá "Instalando dependencias necesarias...". Esto puede tardar un par de minutos dependiendo de tu internet.
2. **Inicio del Servidor**: Una vez termine, el sistema iniciará el servidor local.
3. **Apertura de Pantalla**: Después de unos 5 segundos, se abrirá automáticamente tu navegador (Chrome, Edge, etc.) con el sistema listo para usar.

## ⚠️ Notas Importantes

- **Datos Locales**: El sistema guarda la información **directamente en el navegador** de esa computadora específica. Si quieres llevar los datos de una computadora a otra, deberás usar la función de "Respaldar/Importar" (si está disponible en Configuración) o copiar la base de datos del navegador.
- **No cierres la ventana negra**: Mientras estés usando el sistema, mantén la ventana negra (terminal) abierta. Si la cierras, el sistema dejará de funcionar.
- **Apagado Seguro**: Usa el botón "APAGAR SISTEMA" dentro de la aplicación para cerrar todo correctamente.
